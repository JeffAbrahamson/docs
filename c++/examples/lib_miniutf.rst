Miniutf
=======

miniutf is a C++ implementation of several basic Unicode manipulation functions developed by `Dropbox <https://opensource.dropbox.com/>`_.

The following example shows how to use miniutf to do some format conversions. You can find this example in `the biicode miniutf samples block <http://www.biicode.com/examples/miniutf>`_.

**miniutf.cpp**

.. literalinclude:: /_static/code/cpp/examples/miniutf/miniutf.cpp
   :language: cpp
   :linenos:

Create a block:

.. code-block:: bash

	$ bii init my_project
	$ cd my_project
	$ bii open examples/miniutf


Now you can compile it:

.. code-block:: bash

	$ bii build


You will see next console output after executing the command:

.. code-block:: bash

	$ ./bin/examples_miniutf_miniutf
	OK
