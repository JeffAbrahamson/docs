**bii close**: finish editing published blocks
-----------------------------------------------

You can use **bii close** command to close a block under edition in *blocks* directory.
Usually, to close a block you've **bii open** to edit. 

.. code-block:: bash

	$ bii close USER_NAME/BLOCK_NAME
